Kent OS

Website still in development...

This will document a Debian distribution I'm developing centered on the hardware of a RPi 5 in the shell of an old Zenith Data Systems laptop from the early 90s. The only original hardware in this computer is the repurposed keyboard, everything else is new alongside a lot of 3d printing.

Some unique components to this build:

  • The keyboard is repurposed with a Teensy 4.0 board
  • The "Mouse" uses a Pro Micro board w/ a rotary encoder for scrolling and a PSP joystick for mouse control
  • A 4-digit / 7 Segment display to monitor the battery
  • A control panel on the front w/ ~12 buttons, 7 LEDs, 5 slide switches, and 3 toggle switches
  • A 0.96" pop-out OLED for a variety of diagnostic, network, and other info
  • A DIY 12V battery pack

External interfaces currently include:

  • Ethernet
  • Micro USB plugin for a direct connection to the Pi's UART debug port
  • 3.5mm audio jack (a speaker is also built-in, as well as a slide pot. for volume control)
  • An external HDMI port for connecting multiple displays
  • 2x USB 2.0, 1x USB 3.0
  • A full size SD connecter w/ a built in micro SD card adapter
